NASA Exoplanets

This is a unique infographic visualizing the NASA exoplanets dataset. It showcases the segregation of planets for easier and convinient planet info retrival. There is a central axis that represents 10 years time from 2004 - 2013. There is a custom beeswarm plot on either size of the axis. The left and right side of axis represent the planet type - Gas Giants or Neptune-like respectively. The data shown below consists of 723 planets (datapoints) belonging to the 2 planet types. Each point encodes the exoplanet size and brightness in comparison to a known planet. To know more about a planet use the on-hover tooltip or click on a planet to read more on wikipedia. The infographic shows that year after year we discover more exoplanets, however the number of gas giants discovered every year are always more than the number of neptune like planets. Attributes Used:
  • Planet name
  • Planet type (visualized)
  • Discovery year (visualized)
  • Planet's size (visualized)
  • Planet's brightness (visualized)
  • Distance (lightyears)
Marks:
  • Point mark (circle) where each point represents a planet
Channels:
  • Circle size represent the planet size
  • Circle's Border size (in yellow) represents the planet's brightness
  • Circle horizontal position (left or right of central axis) represents planet type
  • Circle's vertical position represents the discovery year (as per the Y axis)
  • Circle's color represents the planet type (Gas Giant or Neptune-like)
  • Circle's neighbors represent the similar type of planets discovered in the same year
Functionality:
  • Toolip - to hover over any datapoint and find out more about the planet
  • As the neptune-like planets are much smaller than the gas giants, I have provided a button to toggle between their original size (on the same scale) and a more comprehensible size (on a larger scale) for better comparison
